# Memory Snap Play Store Copy

## App Name

Memory Snap

## Short Description

짧은 순간을 기억하고 맞히는 모바일 기억력 미니게임

## Full Description

Memory Snap은 짧은 시간 동안 화면을 관찰하고 기억한 뒤 정답을 맞히는 모바일 기억력 미니게임입니다.

카드 위치 기억, 빛나는 순서 기억, 사라진 물건 찾기 등 여러 기억력 챌린지를 빠르게 플레이할 수 있습니다. 라운드가 올라갈수록 기억해야 할 정보가 늘어나고 제한 시간이 줄어들어, 짧지만 집중도 높은 플레이를 제공합니다.

주요 기능:

- 카드 위치 기억 모드
- 순서 기억 모드
- 사라진 물건 찾기 모드
- 최고 라운드 기록
- 코인과 이어하기 기능
- 모바일 화면에 맞춘 세로형 플레이

Memory Snap은 복잡한 조작 없이 바로 시작할 수 있는 가벼운 기억력 게임입니다. 잠깐의 쉬는 시간에 집중력과 관찰력을 테스트해보세요.
