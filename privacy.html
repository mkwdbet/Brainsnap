Memory Snap

개인정보처리방침

시행일: 2026년 7월 14일

1. 수집하는 정보

Memory Snap은 계정 기능과 기록 저장 기능을 제공하기 위해 다음 정보를 수집할 수 있습니다.

  • 사용자가 입력한 사용자 ID
  • 비밀번호를 직접 저장하지 않고, 인증을 위한 암호화된 비밀번호 정보
  • 게임 최고 기록, 코인 수, 진행 데이터
  • 서비스 오류 확인을 위한 기본 접속 로그

2. 이용 목적

수집한 정보는 로그인, 계정별 기록 저장, 게임 진행 데이터 동기화, 서비스 안정성 개선을 위해 사용됩니다.

3. 제3자 제공

Memory Snap은 법령상 필요한 경우를 제외하고 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다.

4. 보관 및 삭제

계정 정보와 게임 기록은 서비스 제공에 필요한 기간 동안 보관됩니다. 사용자가 삭제를 요청하면 확인 절차 후 합리적인 기간 내 삭제합니다.

5. 아동 개인정보

Memory Snap은 일반 사용자를 대상으로 하는 기억력 미니게임입니다. 만 14세 미만 사용자가 서비스를 이용하는 경우 보호자의 지도가 필요합니다.

6. 문의

개인정보 관련 문의는 출시 전 등록되는 공식 문의 이메일 또는 스토어 등록 정보의 개발자 연락처로 접수할 수 있습니다.
